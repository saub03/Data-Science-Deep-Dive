## 🛠️ 협업 규칙 (Convention)

### 1. 커밋 메시지 (Commit Message)
- `feat`: 새로운 코드/학습 내용 추가
- `docs`: README, 학습 노트 등 문서 수정
- `fix`: 코드 오류 수정
- `refactor`: 코드 가독성 개선 (기능 변경 없음)
- `chore`: 패키지 추가 등 기타 작업
- **예시**: `feat: W1 NumPy 배열 연산 기초 실습`

### 2. 파일 네이밍 (File Naming)
- **형식**: `[날짜]_[주차]_[주제].ipynb`
- **예시**: `260302_W1_NumPy_Basics.ipynb`
- 특수문자나 공백은 지양하고 `_`를 사용하여 연결합니다.
- 선행 학습한 내용의 파일은 `[날짜]_[주차]_[주제]_P.ipynb` 뒤에 P를 붙여서 표시합니다.
