Git 명령어 치트시트
파일 생성 및 깃초기화
mkdir 디렉터리명
touch 파일명       
git init
상태 확인
git status
git log --oneline --branches --graph
git diff
스테이징, 커밋
git add 파일명
git add .
git commit -m "message"
git commit -am "message"
작업 되돌리기
git restore 파일명                   작업트리 변경사항 되돌리기
git restore --staged 파일명    스테이징 되돌리기
git reset HEAD^                     최신 커밋 취소, 언스테이징 -> 작업트리
git reset --soft HEAD^           커밋 취소, staged 상태로
git reset --haed 커밋해시        해당 커밋을 최신 커밋으로(앞의 커밋 모두 삭제)
git revert 커밋해시                 해당 커밋을 최신 커밋으로(앞의 커밋 이력 남겨둠)
브랜치 분기, 병합
git branch
git branch 생성할 브랜치명
git switch 브랜치명
git merge 브랜치명
git branch -d 브랜치명           병합 끝난 브랜치 삭제
git cherry-pick 해시
github 연결
git remote add origin URL
git remote -v
git push -u origin main
git push
git pull origin main
